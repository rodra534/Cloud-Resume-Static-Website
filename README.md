# Cloud-Resume-Static-Website

To design, develop, and deploy a secure, highly available static website that serves as a professional portfolio and interactive digital resume. This project involved designing, developing, and deploying a secure, highly available static website that serves as a professional portfolio and interactive digital resume. The goal was to showcase my ability to leverage core AWS services to host modern web content while maintaining high standards for security and performance.

### Technical Architecture 
* Hosting: Utilized Amazon S3 for cost-effective, scalable static website hosting.
* Content Delivery: Implemented Amazon CloudFront to ensure low-latency global delivery and enhanced security via HTTPS.
* DNS Management: Configured Amazon Route 53 to manage domain routing and ensure reliable site accessibility.
* Infrastructure & Automation: Developed the site using modern HTML/CSS with an integrated visitor counter, powered by a custom API built     on AWS Lambda, allowing for real-time engagement tracking.

### Key Contributions
* Infrastructure as Code & Deployment: Managed the entire deployment lifecycle, including setting up automated content invalidation through   CloudFront to ensure updates are reflected instantly.
* Operational Security: Applied best practices for secure web hosting, ensuring strict control over bucket access and data transmission.
* Professional Branding: Created a centralized digital hub that integrates professional history, certifications, and live technical proof-    of-work, serving as a dynamic alternative to traditional static resumes.

### Lessons Learned 
* Gained practical experience in managing DNS records via Route 53 to ensure seamless domain integration.
* Deepened my understanding of CloudFront behaviors and the necessity of cache invalidation when deploying updates to a static environment.
* Successfully implemented a serverless component (Lambda) to add dynamic functionality to a static site.
